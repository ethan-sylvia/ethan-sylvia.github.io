CSS Explanation
.dropdown .dropdownbtn{ font-family: Roboto, 'sans-serif'; font-size: 16px; border: none; outline: none; color: #1A1B1F; padding: 14px 16px; background-color: white; } This is the CSS used to style the dropdown button so that it looks like a normal link on the navbar.
.dropdown-content{ display: none; position: fixed; background-color: #f9f9f9; box-shadow: 0px 8px 16px 0px rgba(0,0,0,0.2); z-index: 1; } This is the CSS used to style the container of the links. I made the background of the container a light grey color so it is distinct from the white webpage. The position must be fixed so when the hover state is initiated the container stays "fixed" in it's position underneath the button.
.dropdown:hover .dropdown-content { display: block; } This sets the display so the link container is visible once a user hovers over the button
